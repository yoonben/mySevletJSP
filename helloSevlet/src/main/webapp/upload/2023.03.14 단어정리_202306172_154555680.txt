기계어
컴퓨터(운영체제)가 이해하고 실행할 수 있는 0과 1로 이루어진 코드를 말함니다.

프로그래밍 언어
사람이 기계어를 이해하는 것을 매우 어렵기 때문에 사람의 언어와 기계어의 다리 역할을 합니다.

소스파일
프로그래밍 언어로 작성된 파일을 말합니다.

컴파일
소스 파일을 기께어로 번역하는 것을 말합니다. 이 역할을 담당하는 소프웨어를 컴파일러라고 합니다.

JDK
자바 개발 도구의 줄임말로, 자바로 프로그램을 개발 할 수 있는 실행 환경과 개발 도구등을 제공합니다.

환경 변수
운영체제가 실행하는 데 필요한 정보를 제공하는 변수를 말합니다. JDK를 설치한 후 명령 라인에서 컴파일러(javac)와 실행(java)
명령어를 사용하려면 JAVA-HOME 환경 변수를 등록하고 Path 환경변수를 수정하는 것이 좋습니다.

이클립스
무료로 사용할 수 있는 오픈 소스 통합 개발 환경입니다. IDE란 프로젝트 생성,자동 코드 완성, 디버깅 등과 같이 개발에 필요한
여러 가지 기능을 통합적으로 제공해주는 툴을 말합니다.

워크스페이스
이클립스 실행과 관련된 메타 데이터와 프로젝트 폴더가 저장되는 폴더를 말합니다.

뷰
이클립스 내부에서 사용되는 작은 창을 말합니다.

퍼스펙티브
프로젝트를 개발할 때 유용하게 사용할 수 있는 뷰들을 묶어 이름을 붙여 놓은 것을 말합니다.

src
소스파일 저장 공간(.java)

바이트 코드 파일(Hello.class)
자바 소스 파일을 javac 명령어로 컴파일한 파일을 말합니다.

bin
소스가 컴파일되면서 저장하는 공간(.class)

JVM (자바 가상 기계)
자바 가상 기계는 바이크 코트 파일을 운영체제를 위한 완전한 기계어로 번역하고 실행하는 역학을 합니다.
JVM은 명령어에 의해 구동됩니다.

메서드
어떤 일을 처리하는 실행문을 모은 블록

모듈
외부에서 재사용할 수 있는 패키지들은 묶은 것, 이클립스의 프로젝트는 하나의 모듈을 개발하는 것

카엘 표기범
첫문자는 영어 소문자로 시작하되, 다른 단어가 붙을 경우 첫 문자를 대문자로 합니다.

디렉토리
디렉토리는 컴퓨팅에서 파일을 분류하기 위해 사용하는 이름공간이다.
