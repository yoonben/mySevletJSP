필드
클래스 블록 안에 바로 작성되는 변수, 속성을 정의해준다.

변수
값을 저장할 수 있는 메모리 번지에 붙인 이름. 변수를 통해 프로그램은 메모리 번지에 값을 저장하고 읽을 수 있다.

전역변수

지역변수

변수 사용 범위
변수는 자신이 선언된 위치에서 자신이 속한 블록 내부까지만 사용이 가능하고 밖에서는 사용할 수 없다.

리터럴
소스 코드에서 프로그래머에 의해 직접 입력된 값

타입 변환
한 변수값을 타입이 다른 변수에 복사해서 저장할 때 해당 타입으로 변환

자동 타입 변화
값의 허용 범위가 작은 타입이 큰 타입으로 저장될 경우(char타입은 int만 자동변환되고 유니코드 갑ㅅ으로 저장된다)
bytr < short < int <long < float < double

강제 타입 변화
강제로 타입을 변환하는 것. 값의 허용 범위가 큰 타입을 허용 범위가 작은 타입으로 쪼개어서 저장하는 것을 말함

문자열 결합 연산
문자열과 + 연산을 하면 다른 피연산자도 문자열로 변환되어 문자열 결합이 발생함

interger.parseInt() (중요*****)
문자열을 정수 int 타입으로 변환

%s 문자열, %d 정수, %f 실수

Scanner
